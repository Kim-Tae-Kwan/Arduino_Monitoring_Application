# Arduino Monitoring Application
* 아두이노를 통한 조도 센서, 온습도의 데이터를 실시간으로 모니터링 하는 프로그램입니다.
* 아두이노에서 Serial 통신으로 실시간 데이터를 전송하여 C# GUI로 Visualization 하며 MySQL으로 데이터 관리가 가능합니다.
